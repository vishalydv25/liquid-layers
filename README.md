# Liquid Layers 🎨

Liquid Layers is a responsive e-commerce web application built for a resin art brand that sells handmade products like coasters, trays, keychains, wall décor, and personalized gifts.

The idea behind this project was to create a shopping experience that feels simple and smooth. Users can browse products, manage their cart and wishlist, place orders, and keep track of their purchases, all through a clean and easy-to-use interface.

---

## 🌟 Features

- Responsive design for desktop and mobile devices
- User authentication (Sign Up and Login)
- Browse products by category
- Search and filter products
- Product details page
- Shopping cart
- Wishlist
- Address management
- Checkout process
- Order history
- Protected routes
- Toast notifications
- Mock backend powered by MirageJS

## 🚀 Getting Started

### Prerequisites

Before running the project, make sure you have the following installed:

- Node.js
- npm or Yarn

---

### Installation

Clone the repository.

```bash
git clone https://github.com/YOUR_USERNAME/liquid-layers.git
```

Move into the project folder.

```bash
cd liquid-layers
```

Install the required dependencies.

```bash
npm install
```

or

```bash
yarn install
```

Start the development server.

```bash
npm start
```

or

```bash
yarn start
```

Once everything is running, open your browser and visit:

```text
http://localhost:3000
```
